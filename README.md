# AirBnB-Predictions

![Airbnb](https://media.itsnicethat.com/original_images/563b992b7fa44cff9d001763.gif)


## Objetivo:

El objetivo de este proyecto es conseguir predecir con la mayor fiabilidad posible los precios de Airbnb en Amsterdam.


## DOCUMENTOS:

La información está en CSVs alojados en la carpeta "data" ofrecidos a través de un concurso de Kaggle.

- airbdata: es el jupyter notebook donde realizo tanto la limpieza de datos como los modelos predictivos. 

- train.csv: son los datos con los que entrenaremos el modelo predictivo.

- test.csv: una vez consigamos el modelo, lo testearemos con este archivo intentando predecir los precios.

- sample.csv: son las id para los precios que hemos conseguido. El archivo que subiremos a la competicion de kaggle será con este formato (dos clumnas de id y precio).

- Los archivos que empiezan por prediccion seguido de tres letras son los archivos "sample" que subía como submission y las últimas 3 letras del nombre corresponden al modelo utilizado para la predicción.
